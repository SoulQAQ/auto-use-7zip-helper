# 自动打包器

自动将文件/文件夹加密打包成双层压缩包的工具。

## 功能
- 支持拖拽文件/文件夹
- 自动生成随机密码
- 7z加密压缩 + zip存储打包
- 支持打包后伪装为媒体/文档文件（PNG/JPG/MP3/MP4/PDF）
- 可选删除源文件

## 伪装功能说明

打包流程：

`源文件 -> 加密7z -> 密码文件+说明文本 -> zip -> 伪装输出`

伪装类型：
- `PNG`：支持动态随机生成载体，也可指定自定义载体
- `PDF`：支持动态随机生成载体，也可指定自定义载体
- `JPG / MP3 / MP4`：需要载体文件（可每次手动选择，或在设置中配置默认模板）

载体优先级：
1. 本次打包手动选择的载体文件
2. 设置中已配置的默认模板（保存在 `media-templates/`）
3. 动态生成（仅 `PNG/PDF` 支持）

提示：
- 伪装输出文件可使用 7-Zip 进行解压
- 当当前伪装类型没有可用载体时，开始打包按钮会自动禁用

## 使用方法
1. 运行 `setup.bat` 安装环境
2. 运行 `start.bat` 启动程序
3. 运行 `build.bat` 打包成EXE
